# 像素风情绪小屋 v2 实验版 - 产品需求文档

## 项目概述

基于稳定版v1，进行全方位大幅升级。保留核心概念（情绪房间+猫咪+音乐），但在美术品质、互动深度、挂机玩法三个维度实现质的飞跃。

## 核心升级方向

### 一、美术大幅升级

#### 1.1 像素资源升级
- **房间场景**：从当前简单CSS方块升级为高精度像素画风格，使用Canvas全场景渲染
- **家具道具**：每个家具从单色方块升级为16x16/32x32精细像素画，带阴影和高光
- **小猫**：从2帧升级为8帧流畅动画（走路、坐下、睡觉、玩耍、伸懒腰、洗脸）
- **窗户场景**：动态天气效果（雨滴、雪花、星星闪烁、云朵飘动）
- **光影系统**：台灯光晕、月光投射、情绪色调渲染

#### 1.2 视觉特效
- **粒子系统升级**：萤火虫、花瓣飘落、星星闪烁、雨滴涟漪
- **情绪氛围**：每种情绪有独特的全屏氛围效果（暖光/冷调/梦幻/暗夜/电光）
- **昼夜循环**：真实的天空渐变，窗户景色随时间变化
- **CSS动画**：家具微动、窗帘飘动、植物摇曳

#### 1.3 UI美化
- **侧边栏**：像素风窗口框架，半透明毛玻璃效果
- **播放器**：唱片机3D旋转效果，频谱可视化
- **按钮**：像素风3D凸起按钮，按下有弹跳反馈
- **字体**：保持像素风但增加更多装饰性排版

### 二、互动逻辑大幅优化

#### 2.1 家具互动升级
- **书桌**：点击后小猫跳上桌，显示迷你像素画/日记
- **掌机**：点击后进入迷你像素游戏（贪吃蛇/打砖块）
- **手办架**：点击旋转展示，收集系统
- **零食盒**：点击喂猫，增加好感度
- **爱心盒**：打开获得随机装饰品
- **盆栽**：浇水成长系统（种子→发芽→开花）
- **吊饰**：风吹摆动，点击许愿
- **玩偶**：点击触发不同动画（跳跃/旋转/发光）

#### 2.2 猫咪系统升级
- **好感度系统**：0-100，影响猫咪行为
- **猫咪状态**：饥饿、开心、困倦、好奇
- **互动方式**：抚摸（长按）、喂食、逗玩、拍照
- **猫咪AI**：根据情绪和好感度自主行动
- **猫咪礼物**：好感度足够时猫咪会带来礼物

#### 2.3 情绪系统升级
- **5种情绪**保持，但每种情绪有3个渐变阶段
- **情绪触发**：不再只是按钮，可通过互动自然触发
- **情绪联动**：猫咪行为、天气、时间共同影响情绪

### 三、挂机玩法系统

#### 3.1 星光货币
- **获取方式**：
  - 每分钟自动获得1星光
  - 猫咪带来礼物获得5-20星光
  - 完成小任务获得10-50星光
  - 挂机每10分钟触发一次"猫咪事件"
- **用途**：购买家具、装饰品、解锁新猫咪皮肤

#### 3.2 挂机事件
- **猫咪探险**：猫咪离开房间，回来带回随机物品
- **访客系统**：偶尔有其他小动物来访（小鸟、蝴蝶、萤火虫群）
- **天气事件**：暴风雨、流星雨、彩虹
- **梦境系统**：长时间挂机后猫咪入睡，显示梦境画面

#### 3.3 成就系统
- **收集成就**：收集所有装饰品、家具升级
- **时间成就**：累计在线时间里程碑
- **互动成就**：与猫咪互动次数、喂食次数
- **音乐成就**：听歌时长、切换歌单次数

#### 3.4 家具升级
- **书桌**：Lv1普通→Lv2带台灯→Lv3带电脑→Lv4全装备
- **唱片机**：Lv1基础→Lv2带音箱→Lv3DJ台
- **盆栽**：Lv1种子→Lv2发芽→Lv3开花→Lv4果实
- **猫窝**：Lv1纸箱→Lv2垫子→Lv3豪华猫窝

## 技术约束

- 保持单HTML文件架构
- 像素资源通过Canvas程序化生成或外部像素画API
- 网易云音乐系统沿用v1本地API方案
- 所有数据存储在localStorage
- 兼容Chrome/Edge/Firefox最新版

## 开发优先级

1. P0：Canvas全场景渲染 + 精细像素画资源
2. P0：猫咪8帧动画 + 好感度系统
3. P1：挂机星光系统 + 猫咪事件
4. P1：家具升级系统
5. P2：迷你游戏（掌机）
6. P2：成就系统
7. P3：访客系统 + 梦境系统
